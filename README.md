# GMEMC 사이트 리뉴얼

[https://gmemc.co.kr/](https://gmemc.co.kr/) 리뉴얼 프로토타입입니다.

## 포함 내용

- `research/` — 30인 패널 분석, 개선안
- `src/` — Next.js 리뉴얼 사이트 (HOME / COMPANY / PROJECT / CONTACT)
- `public/assets/` — 기존 업로드 자산 + AI 보완 히어로

## 실행

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:3000

## 유지 계정·자산

- 지명원 Drive 링크 그대로 연결
- 로고·프로젝트·CEO·지도 등 기존 업로드 이미지 사용
- GitHub 푸시/커밋은 보유 계정으로 진행 (`gh auth login` 후 push)
